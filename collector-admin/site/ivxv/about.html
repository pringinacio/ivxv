Versioon
Elektroonilise hääletamise infosüsteem IVXV.
Kogumisteenuse haldusteenus.
Versioon: 1.9.10
Kasutajaliides
Haldusteenuse kasutajaliidese funktsioonid:
  • Kogumisteenuse seisundi jälgimine;
  • Valijate ja valikute nimekirjade haldus;
  • Statistika vahendamine e-hääletuse kulgemise kohta;
  • Haldusteenuse kasutajate haldus;
  • Kogumisteenuse koosseisu haldus;
  • Allalaadimised (e-valimiskast, hääletamise detailstatistika);
  • Kogumisteenuse halduse sündmuste logi kuvamine.
Masinloetavad andmed
Masinloetavad andmed (JSON):
  • Kasutajaliidese kontekst;
  • Kogumisteenuse olek;
  • Hääletuse statistika;
  • Ringkondade nimekiri;
  • Haldusteenuse sündmuste logi;
  • Allalaadimiseks koostatud e-valimiskastide seisund.
Kasutajaliidese ehitamiseks on kasutatud BootsTrap teemat SB Admin 2.
